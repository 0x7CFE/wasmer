wasmer_clif_backend

[−][src]Module wasmer_clif_backend::relocation

The relocation package provide two structures: RelocSink, TrapSink. This structures are used by Cranelift when compiling functions to mark any other calls that this function is doing, so we can "patch" the function addrs in runtime with the functions we need.
