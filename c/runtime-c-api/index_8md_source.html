1 # Wasmer Runtime C API
2 
3 [Wasmer] is a standalone [WebAssembly] runtime, aiming to be fully
4 compatible with WASI, Emscripten, Rust and Go. [Learn
5 more](https://github.com/wasmerio/wasmer).
6 
7 The `wasmer-runtime-c-api` crate exposes a C and a C++ API to interact
8 with the Wasmer runtime. This document is the index of its
9 auto-generated documentation.

14 # Usage
15 
16 Since the Wasmer runtime is written in Rust, the C and C++ API are
17 designed to work hand-in-hand with its shared library. The C and C++
18 header files, namely [`wasmer.h`] and [`wasmer.hh`] are documented
19 here. Their source code can be found in the source tree of this
20 crate. They are automatically generated, and always up-to-date in this
21 repository. The C and C++ header files along with the runtime shared
22 libraries (`.so`, `.dylib`, `.dll`) can also be downloaded in the
23 Wasmer [release page].
24 
25 [`wasmer.h`]: ./wasmer_8h.html
26 [`wasmer.hh`]: ./wasmer_8hh.html
27 [release page]: https://github.com/wasmerio/wasmer/releases
28 
29 Here is a simple example to use the C API:
30 
31 ```c
32 #include <stdio.h>
33 #include "../wasmer.h"
34 #include <assert.h>
35 #include <stdint.h>
36 
37 int main()
38 {
39  // Read the Wasm file bytes.
40  FILE *file = fopen("sum.wasm", "r");
41  fseek(file, 0, SEEK_END);
42  long len = ftell(file);
43  uint8_t *bytes = malloc(len);
44  fseek(file, 0, SEEK_SET);
45  fread(bytes, 1, len, file);
46  fclose(file);
47 
48  // Prepare the imports.
49  wasmer_import_t imports[] = {};
50 
51  // Instantiate!
52  wasmer_instance_t *instance = NULL;
53  wasmer_result_t instantiation_result = wasmer_instantiate(&instance, bytes, len, imports, 0);
54 
55  assert(instantiation_result == WASMER_OK);
56 
57  // Let's call a function.
58  // Start by preparing the arguments.
59 
60  // Value of argument #1 is `7i32`.
61  wasmer_value_t argument_one;
62  argument_one.tag = WASM_I32;
63  argument_one.value.I32 = 7;
64 
65  // Value of argument #2 is `8i32`.
66  wasmer_value_t argument_two;
67  argument_two.tag = WASM_I32;
68  argument_two.value.I32 = 8;
69 
70  // Prepare the arguments.
71  wasmer_value_t arguments[] = {argument_one, argument_two};
72 
73  // Prepare the return value.
74  wasmer_value_t result_one;
75  wasmer_value_t results[] = {result_one};
76 
77  // Call the `sum` function with the prepared arguments and the return value.
78  wasmer_result_t call_result = wasmer_instance_call(instance, "sum", arguments, 2, results, 1);
79 
80  // Let's display the result.
81  printf("Call result: %d\n", call_result);
82  printf("Result: %d\n", results[0].value.I32);
83 
84  // `sum(7, 8) == 15`.
85  assert(results[0].value.I32 == 15);
86  assert(call_result == WASMER_OK);
87 
88  wasmer_instance_destroy(instance);
89 
90  return 0;
91 }
92 ```

94 # Testing
95 
96 Tests are run using the release build of the library. If you make
97 changes or compile with non-default features, please ensure you
98 rebuild in release mode for the tests to see the changes.
99 
100 The tests can be run via `cargo test`, such as:
101 
102 ```sh
103 $ cargo test --release -- --nocapture
104 ```
105 
106 To run tests manually, enter the `lib/runtime-c-api/tests` directory
107 and run the following commands:
108 
109 ```sh
110 $ cmake .
111 $ make
112 $ make test
113 ```
